Une application web complète de gestion de pharmacie permettant d’administrer efficacement les stocks de médicaments, les utilisateurs et les factures.
Le système facilite le suivi des produits pharmaceutiques, la gestion des ventes et l’organisation des opérations quotidiennes de la pharmacie.

Fonctionnalités principales
-Gestion des utilisateurs et des rôles (administrateur, pharmacien, caissier)
-Gestion du stock de médicaments
-Suivi des entrées et sorties de produits
-Gestion des factures et des ventes
-Alertes sur les produits en rupture ou proches de la date d’expiration
-Historique des opérations
-Tableau de bord statistique
-Recherche rapide de produits et de factures
-Sécurisation des accès et authentification

Objectifs du projet
-Automatiser la gestion du stock pharmaceutique
-Réduire les erreurs de gestion manuelle
-Optimiser le suivi des ventes et des achats
-Faciliter le contrôle des médicaments disponibles et expirés

Technologies utilisées
Langage : JavaFx
Base de données : MySQL
API REST
Git & GitHub pour le versioning
